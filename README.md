# 坦克大战 - Tank Battle

一个经典风格的 HTML5 坦克大战游戏，包含现代化的视觉效果和丰富的游戏玩法。

## 🎮 [在线游玩](https://hgwly.github.io/tank/)

## 游戏截图

![游戏画面](screenshot.png)

## 操作方式

| 按键 | 功能 |
|------|------|
| `W` `A` `S` `D` | 移动坦克 |
| `↑` `←` `↓` `→` | 移动坦克（方向键） |
| `空格` | 发射子弹 |
| `P` | 暂停游戏 |
| `M` | 静音/取消静音 |

## 游戏特色

### 🎯 核心玩法
- **消灭敌方坦克**：摧毁所有出现的敌人以进入下一关
- **保卫基地**：保护地图底部的蓝色鹰徽基地不被摧毁
- **收集道具**：击毁敌人有概率掉落特殊道具

### ⭐ 道具系统

| 道具 | 效果 |
|------|------|
| ⭐ 星星 | 升级武器，提高射速和威力 |
| ❤️ 绿心 | 获得额外生命 |
| 🕐 蓝钟 | 冻结时间，敌人暂停行动 5 秒 |
| 🛡️ 紫盾 | 获得 10 秒无敌护盾 |

### 👾 敌人类型

| 类型 | 特点 |
|------|------|
| 普通坦克 | 红色，标准速度和血量 |
| 快速坦克 | 橙色，移动速度更快 |
| 重装坦克 | 紫色，血量更高（3 点） |
| BOSS 坦克 | 深红色，高血量（5 点），第 4 关起出现 |

### 📊 游戏元素

| 元素 | 说明 |
|------|------|
| 🟩 绿色坦克 | 玩家坦克 |
| 🟥 红色坦克 | 敌方坦克 |
| 🟧 橙色砖块 | 可破坏的墙壁（2 点血量） |
| ⬜ 灰色钢块 | 不可破坏的墙壁 |
| 🔵 蓝色鹰徽 | 基地（被摧毁则游戏结束） |

## 游戏机制

### 得分系统
- 击毁普通坦克：+100 分
- 击毁重装坦克：+300 分
- 击毁 BOSS 坦克：+500 分
- 完成关卡：+500 分

### 关卡系统
- 每关需要消灭 10 + 关卡数×3 个敌人
- 关卡越高，敌人数量越多、生成速度越快
- 第 2 关起出现快速坦克
- 第 3 关起出现重装坦克
- 第 4 关起出现 BOSS 坦克

## 技术栈

- **HTML5 Canvas** - 游戏渲染
- **Vanilla JavaScript** - 游戏逻辑（ES6+ Classes）
- **CSS3** - 界面样式和动画
- **Web Audio API** - 合成音效

## 本地运行

1. 克隆仓库：
```bash
git clone https://github.com/hgwly/tank.git
cd tank
```

2. 直接在浏览器中打开 `index.html` 文件

或使用本地服务器：
```bash
# Python 3
python -m http.server 8000

# Node.js (npx)
npx serve
```

然后访问 `http://localhost:8000`
