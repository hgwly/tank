# 坦克大战 (Tank Battle)

一个经典的 HTML5 坦克大战游戏。

## 在线演示

[点击这里](https://hgwly.github.io/tank/) 在线游玩。

## 游戏说明

### 操作方式
- **WASD** 或 **方向键**：移动坦克
- **空格键**：发射子弹
- **P 键**：暂停游戏

### 游戏目标
- 消灭所有敌方坦克
- 保护你的基地（地图底部的蓝色圆形）
- 获得更高的分数

### 游戏元素
- 🟩 绿色坦克：玩家
- 🟥 红色坦克：敌人
- 🟧 橙色砖块：可破坏的墙壁
- ⬜ 灰色钢块：不可破坏的墙壁
- 🔵 蓝色圆形：基地（需要保护）

### 得分
- 击毁敌方坦克：+100 分
- 每 500 分进入下一关

## 技术栈
- HTML5 Canvas
- 原生 JavaScript
- CSS3

## 本地运行

直接双击打开 `index.html` 文件即可在浏览器中运行游戏。
